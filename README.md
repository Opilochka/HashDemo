# HashDemo

Демонстрационный проект на C#, реализующий:

- **Ручную реализацию алгоритма SHA-256** (без использования `System.Security.Cryptography`)
- **Упрощённую хэш-функцию** с демонстрацией коллизии

Проект создан в учебных целях для изучения криптографических хэш-функций и работы с документацией.

## Сборка и запуск

1. Убедитесь, что установлен [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0).
2. Клонируйте или откройте проект.
3. Соберите и запустите:

```powershell
cd HashDemo
dotnet run
```

Проект содержит документацию. HashDemo\HashDemo\_site\api\HashDemo.html

## Программа запустит интерактивный режим:

Ввод строк для хэширования

Проверка коллизий в упрощённой функции


## Примеры
SHA256("abc") → ba7816bf8f01cfea414140de5dae2223b00361a396177a9cb410ff61f20015ad

SimpleHash("ab") == SimpleHash("ba") → коллизия найдена
